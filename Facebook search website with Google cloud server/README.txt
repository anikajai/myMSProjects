Created Responsive website for Facebook search on users/pages/groups/events/places using AngulaJS and Bootstrap.
It has feature to add/remove records to local favorite list. also, every record details consist of posts and albums.
User can even post something to facebook from here.

The Facebook garph API is used and the PHP code to access that is running on Google Cloud server.
